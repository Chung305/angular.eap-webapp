Early Access Program

The Purpose of this program is to be able to still adapt our starting user base and customer base while simultaneously taking the pressure off of delivering a fully featured application at the inception of this program on the business. The release for the Early Access Program (EAP) is set for the first week of april. After production release for the EAP of Overseas Basketball Connection Web App, customers that signed up will be able to sign in using their supplied access code or login credentials for the first version of the released software as is at launch. During EAP, users will be asked to give feedback on improvements as well as participate in marketing events coordinated by the business. This will give us a good understanding of the trajectory of the path to completion and what we need to consider when furthering the application. EAP releases will range in versions from v0.0.1 to v0.9.9. Each change will be released on their respective schedules.

Post EAP

The EAP is only intended to last a couple of months until restrictions are taken off and the application is open to the public for consumption. When the application goes fully live in a public production environment, users (players/teams) will be able to sign up for the service normally through the provided channels of authentication. This release will be considered “OBC Web App v1.0”.
